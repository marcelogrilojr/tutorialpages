Tutorial de como hosperdar um site no GithubPages

Sem o GIT

  1. Criar um respositório no github
    • Marcar como público, não marcar as outras opções (readme, license, gitignore)
  2. Fazer o upload de todos os arquivos e pastas do nosso site
  3. Ir sem settings e procurar no menu à esquerda a opção PAGES
  4. Escolher a branch main para acessar o nosso site

PRONTO!
